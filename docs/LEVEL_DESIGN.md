# 关卡设计：周期-5 极限填充 + 距离插值

## 核心思路

1. 算出 15×15 棋盘上**不出现五连**的黑子极限布局（144 子）
2. 将 144 颗子按**离中心距离从远到近**排序
3. 取前 N 颗 → 即为「N 子难度」的关卡

难度越高 = 黑子越多 = 越靠近中心 = 对玩家威胁越大。

## 极限布局公式

```
cell(r, c) 为黑子  ⟺  (r%5 ≠ c%5) ∧ ((r+c)%5 ≠ 0)
```

等价地，cell(r, c) 为**空格** ⟺ `(r%5 == c%5)` 或 `((r+c)%5 == 0)`。

### 为什么是 144？

| 删除模式 | 条件 | 保护方向 | 删除格数 |
|---------|------|---------|---------|
| A | `r%5 == c%5` | 横向 & 纵向 | 45 |
| B | `(r+c)%5 == 0` | ↘对角 & ↗对角 | 45 |
| A∩B | `r≡0(mod5) 且 c≡0(mod5)` | （重复计） | 9 |

空格 = 45 + 45 − 9 = **81**，黑子 = 225 − 81 = **144**。

### 四方向安全性验证

- **横向**（固定 r，c 递增）：模式 A 保证每 5 列必有一空 ✓
- **纵向**（固定 c，r 递增）：模式 A 保证每 5 行必有一空 ✓
- **↘对角线**（r,c 同增，r+c 每步 +2）：模式 B 保证每 5 步命中 `(r+c)%5==0` ✓
- **↗对角线**（r 增 c 减，r−c 固定）：模式 A 保证每 5 步命中 `r%5==c%5` ✓

## 排序与插值

144 颗子按 `dist = (r−7)² + (c−7)²` **降序排列**（边缘优先）。

## 扩展方式

如需增加关卡，只需在 `counts` 数组中插入新的数值即可。每个关卡自动是上一关的超集（包含关系），无需手工放置。

若需打破单调感，可以：
- 微调排序权重（如加入随机扰动或方向偏好）
- 对特定关卡手动交换几颗子的顺序，制造局部战术点
- 改变排序策略（如按象限轮流填充）
