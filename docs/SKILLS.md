# 技能系统与 Roguelike 成长设计

## 一、核心设计原则

- **极简**：不引入额外的数值资源（没有法力值、没有经验值）
- **潜规则尽量少**：所有规则对玩家可见，不依赖隐藏公式
- **成长感来自积累**：玩家变强是因为手里的技能越来越多、越来越好

---

## 二、技能分类

技能天然分为两大类，这是游戏压力平衡的核心：

### 威胁消除类（减少黑子）

直接移除或转化黑子，降低场上黑子总量。

- 釜底抽薪（移除1黑子）
- 浑水摸鱼（随机移除2黑子）
- 空城计（清除中心3×3区域黑子）
- 破釜沉舟（移除3颗最危险黑子）
- 上屋抽梯（移除最长黑线中间一颗）
- 偷梁换柱（黑子变白子）
- 顺手牵羊（孤立黑子变白子）
- 反客为主（最中心黑子变白子）
- 等等

### 结构调整类（不减少黑子）

移动、放置白子、交换位置，不改变黑子总量。

- 移花接木（移动黑子到指定空位）
- 调虎离山（移动黑子到边缘）
- 乾坤大挪移（交换黑白子位置）
- 无中生有（放置白子）
- 围魏救赵（在黑子群旁放2白子）
- 无懈可击（封堵最长黑线两端）
- 等等

**为什么这样分？**

AI 每回合放 1 颗黑子，玩家每 3 回合才获得 1 张技能卡。只要技能池里有足够的"结构调整类"技能，黑子净流入就是正的。玩家不可能只靠技能无限清场，必须同时靠落白子赢棋。这就是压力来源。

---

## 三、技能稀有度

三个等级控制成长节奏：

| 稀有度 | 标签色 | 何时可获得 | 设计原则 |
|--------|--------|-----------|---------|
| 普通 (common) | 灰色 | 所有关卡 | 单目标、基础效果 |
| 稀有 (rare) | 蓝色 | 第 3 关起 | 智能选目标、组合效果 |
| 史诗 (epic) | 金色 | 第 5 关起 | 多目标、大范围、强力效果 |

**抽卡规则：**
- 每次 3 选 1 时，候选池根据当前关卡序号过滤稀有度
- 关卡 1-2：只出普通
- 关卡 3-4：普通 + 稀有
- 关卡 5+：普通 + 稀有 + 史诗
- AI 生成的技能（第三张）默认为稀有，后期关卡为史诗

---

## 四、Run 结构（Roguelike 闯关）

### 4.1 Run 流程

```
开始 Run
  → 关卡 1（初始黑子少，AI 弱）
    → 获胜 → 奖励选择（3 选 1 技能加入手牌）
  → 关卡 2（黑子增多）
    → 获胜 → 奖励选择
  → ...
  → 关卡 7（BOSS 级，黑子密集，AI 强）
    → 获胜 → Run 通关！

  任何关卡失败 → Run 结束
```

### 4.2 手牌持久化（核心机制）

**这是整个成长系统的唯一核心机制：手牌跨关卡保留。**

- 过关后，手中剩余的技能卡全部带入下一关
- 过关奖励额外获得 1 张技能卡（3 选 1）
- 手牌上限：5 张（与现有一致）
- 关卡内抽卡规则不变（每 3 回合 3 选 1）

**成长曲线示意：**

```
关卡 1 开局：0 张手牌
  → 关卡内通过抽卡获得若干技能
  → 过关时手中有 2-3 张
  → 奖励 +1 张
  → 关卡 2 开局：3-4 张手牌（已有优势）
  → ...
  → 后期关卡开局：接近满手牌，且含稀有/史诗技能
```

玩家自然会在后期积累更多技能，对抗更难的关卡。不需要额外数值系统。

### 4.3 关卡难度递增

关卡序号越大：
- 预置黑子越多
- 黑子布局威胁度越高（更多三连/四连）
- AI 水平越高（random → basic → intermediate）

现有的 10 个关卡已经按此设计。Run 模式下按顺序闯关。

### 4.4 奖励选择

过关后弹出奖励界面，展示 3 张候选技能卡，玩家选 1 张加入手牌：
- 候选池根据下一关的关卡序号决定稀有度范围
- 奖励候选不包含手牌中已有的同名技能（避免重复）

---

## 五、关于弃牌

当前不做弃牌机制。手牌上限 5 张已经是天然约束：
- 抽卡时手牌已满就不再触发抽卡
- 玩家需要主动使用技能腾出空间
- 生存压力（关卡越来越难）会迫使玩家消耗手牌

如果测试发现玩家囤牌问题严重，再考虑加入弃牌环节。

---

## 六、不做的事

为保持轻度，以下机制暂不实现：

- ❌ 技能升级系统
- ❌ 被动遗物/装备
- ❌ 路径选择/分支地图
- ❌ 商店
- ❌ AI 加速放子
- ❌ 法力值/费用系统
- ❌ 技能冷却
